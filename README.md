# PartyRooms - Frontend

## Установка

Чтобы установить проект

```bash
  git clone https://github.com/bodyisthere/PartyRoomsFrontend.git
  cd PartyRoomsFrontend
  npm i
```

## Запуск тестов

Чтобы запустить тесты в консоли:

```bash
  npm run test:unit
```

Чтобы запустить тесты с UI панелью:

```bash
  npm run test:unit:ui
```
